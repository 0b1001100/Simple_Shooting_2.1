ゲーム名:Simple_shooting_2.1
著者:Asynchronous
主に制作した人:0x4C

Simple_shooting_2_1.exeをダブルクリックすれば始まります。


--操作説明--

基本的にはチュートリアルをプレイしたら分かります。


[キーボード]

【W,A,S,D】
移動

【マウス左クリック】
射撃
長押しで連射

【TABキー】
メイン武器切り替え

【Ctrl】
メニュー


[コントローラー]
対応コントローラー:Xbox、DualShock(DualSenseもたぶん使えます)

【左スティック】
移動

【右スティック】
射撃

【□ボタン】
メイン武器切り替え

【△ボタン】
メニュー


--画質の設定--

このゲームは、プレイするパソコンのスペックを考えて初期状態では画質が"低"に設定されています。
自身のパソコンのスペックを見て"高"や"ウルトラ"にするか決めて下さい。
